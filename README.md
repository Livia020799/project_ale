# Project ALE

An interactive mission-planning experience built as a side project and birthday surprise for an aerospace engineering student.
Design a custom space mission by selecting its objective, launch vehicle, and payload, then inspect real 3D spacecraft before initiating the final launch sequence.

Built with vanilla HTML, CSS and JavaScript, featuring real-time 3D models rendered with `<model-viewer>`. Developed with AI-assisted (vibe coding) using ChatGPT-5.

The interface is fully responsive and optimized for both desktop and mobile devices.

You can also create a link to directly host the static webpage with the animation on GitHub. Mine is: https://livia020799.github.io/project_ale/

Tutorial on how to create the link on GitHub

1. Create a GitHub account.
2. Create a new repository.
3. Upload the HTML, CSS, JavaScript files and the `.glb` models.
4. Go to **Settings → Pages**.
5. Select the **main** branch and click **Save**.
6. Refresh the page or come back after a couple of minutes. At the top of the **Pages** section you'll find the link to your static webpage.
